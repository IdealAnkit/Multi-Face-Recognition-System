# Data Directory

This directory stores all data generated by the Multi-Face Recognition Attendance System.

## Structure

```
data/
├── attendance.csv          # Daily attendance records (excluded from Git)
└── enrolled_people/        # Face embeddings and images (excluded from Git)
    └── {ID}_{Name}/
        ├── face_01.jpg to face_20.jpg  # 20 face samples
        ├── thumbnail.jpg               # Profile picture
        ├── embeddings.npy              # Face embeddings (128-D vectors)
        ├── embedding_mean.npy          # Average embedding
        └── meta.json                   # Metadata (name, ID, date)
```

## Privacy Notice

⚠️ **The contents of this directory are excluded from version control** (via `.gitignore`) because:

- Face images and embeddings are **personal/biometric data**
- Attendance records contain **sensitive information**
- Each deployment should maintain its own local data

## Sample Data Format

See `attendance_sample.csv` in the project root for the attendance CSV format.

When you enroll users, their data will be automatically stored in the appropriate structure.
